# Front-end Sistema de Cuidado de Pets

Interface web para gerenciamento de pets, diários, veterinários e agendamentos.

## Funcionalidades

- Cadastro e listagem de pets
- Diário médico dos pets
- Observações
- Cadastro de veterinários
- Agendamento de consultas
- Validação de endereços por CEP

## Como usar

Abra o arquivo `index.html` em um navegador web.

Certifique-se de que as APIs estão rodando nas portas 5000 e 5001.
